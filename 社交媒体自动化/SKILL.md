# 📸 社媒全自动助手 (支持批量 & 珠宝专精版)

这个技能专门用于处理图片（特别是珠宝首饰），支持**批量上传**。它会识别每一张图片的内容，进行针对性的专业修图（珠宝模式/通用模式），并为每一张图生成专属的爆款文案。

## 🛠️ 前置检查
```bash
python -c "import PIL" || pip install pillow
```

## 🚀 工作流程

### 1. 接收图片
询问用户提供图片路径。**支持一次性提供多张图片**（拖入多个文件或提供文件夹路径）。
如果用户提到"珠宝"、"首饰"、"戒指"等关键词，或者图片内容被识别为珠宝，自动启用 **`jewelry` (珠宝模式)**。

### 2. 批量处理循环
为用户提供的一组图片创建一个桌面文件夹：`C:\Users\sai\Desktop\社媒发布_YYYYMMDD_HHMM`。
然后，对**每一张**图片执行以下步骤：

#### A. 视觉分析 (Vision)
使用 `Read` 工具读取图片。
*   **若是珠宝**: 重点识别 **材质** (18K金/铂金/纯银/玫瑰金)、**宝石** (钻石/红蓝宝/翡翠/珍珠)、**款式** (复古/极简/奢华/设计感)、**佩戴部位**。
*   **若是其他**: 分析场景、氛围、主体。

#### B. 生成文案 (Creative Writing)
根据视觉内容，生成两段文案（追加到 `文案汇总.txt` 文件中）：

**💎 小红书风格 (珠宝版)**
*   **标题**: 情绪价值+产品痛点 (e.g., "一眼心动！这也太显白了吧✨", "本命首饰💍戴上就舍不得摘")。
*   **正文**:
    *   强调**细节** (切工、火彩、光泽)。
    *   强调**场景** (约会、通勤、送礼、取悦自己)。
    *   包含材质说明 (e.g., "Au750", "天然钻石")。
    *   大量使用珠宝相关 Emoji (💍💎✨📿🧚‍♀️)。
*   **标签**: #珠宝 #首饰分享 #戒指 #穿搭 #小众设计 #显白

**🟢 朋友圈风格**
*   **风格**: 此时无声胜有声，或者简短的高级感。
*   **示例**: "捕捉一抹指尖的光。✨", "今日份的小确幸。", "属于夏天的光泽感。"

#### C. 智能修图 (Image Processing)
根据图片类型调用脚本。
*   **珠宝模式**: `jewelry` (高锐度、高质感、防过曝)
*   **通用模式**: `general` (鲜艳、明亮)

```bash
# 示例命令 (根据实际文件数循环)
python "C:\Users\sai\.claude\skills\social-media-automator\scripts\image_enhancer.py" "<原图路径>" "<桌面文件夹>\图1_修图后.jpg" "jewelry"
```

### 3. 交付成果
1.  所有修好的图和 `文案汇总.txt` 都保存在桌面文件夹中。
2.  输出一段简报：
    *   "已处理 X 张图片。"
    *   "识别为 [珠宝/通用] 模式。"
    *   "✅ 都在桌面文件夹里了，快去看看吧！"
3.  使用 `explorer` 打开该文件夹。
